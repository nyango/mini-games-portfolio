# NEON COMBO BURST

シンプルで少し中毒性のあるワン画面アーケードゲームです。

## 遊び方

- `index.html` をブラウザで開く
- マウス/指で自機を誘導
- クリック・タップ・Spaceでバーストダッシュ
- バースト中に赤い敵へ体当たりすると撃破＆コンボ
- 青いコアを拾うとバーストゲージが回復
- バーストしていない時に敵へ触れるとゲームオーバー

## モード

### NORMAL

元の緊張感重視のモードです。短いバーストで正確に敵を砕きます。

### 楽々連鎖モード

爽快感と練習しやすさ重視のモードです。

- バースト消費が少ない
- バースト時間が長い
- 敵を倒せる当たり判定が広い
- コンボ受付時間が長い
- 青コアの回復量が多い
- 敵を倒すと周囲の敵を巻き込む連鎖爆発が発生
- 敵に触れてもゲージがあれば緊急バリアで一度助かる

## 攻略法

ゲーム内の「攻略法を見る」から確認できます。基本は以下です。

1. 普段は中央寄りで安全に移動する
2. 敵が直線上に並ぶまで待つ
3. 敵の少し手前でバーストする
4. 撃破後の一瞬スロー中に次の敵へ狙いを合わせる
5. 青コアを拾ってゲージを切らさない
6. 画面端に追い込まれないよう円を描いて逃げる

## ファイル

- `index.html` - 画面
- `style.css` - ネオン風UI
- `game.js` - ゲーム本体

外部ライブラリやビルドは不要です。
