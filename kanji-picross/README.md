# 漢字発掘ピクロス

石板を削って隠された漢字を掘り出す、10x10のピクロス/ノノグラムです。

## 遊び方

- 行と列の数字を見て、掘るべきマスを推理します。
- 「掘る」モードで正解マスをクリック/タップすると金色の線が現れます。
- 「印」モード、または右クリックで「ここは掘らない」印を付けられます。
- 間違って空白マスを掘るとミスになります。
- 正解マスをすべて掘ると漢字発掘完了です。

## 操作

- クリック/タップ: 現在のモードでマス操作
- 右クリック: 印を付ける/外す
- `D` or `1`: 掘るモード
- `M` or `2`: 印モード
- `H`: ヒント
- `R`: リセット

## パズル

山、川、木、日、月、火 の6問入りです。
